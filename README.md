# 💧 Machine Learning-Based Water Potability Prediction with Geospatial Insights (2025)

## 🚀 Features
- **Potability Prediction:** Determines if water is safe for drinking.
- **Machine Learning Models:** Random Forest and Gradient Boosting for accurate predictions.
- **Geospatial Visualization:** Interactive maps for potability insights.
- **Performance Metrics:** ROC-AUC, accuracy, and confusion matrix for evaluation.
- **User-Friendly Interface:** Simple and clear output.

## 🛠️ Tech Stack
- **Frontend:** React + Vite  
- **CSS Framework:** Tailwind CSS  
- **Others:** Axios (for API calls), React Router  

## 📖 Getting Started

### Prerequisites
- Node.js installed  
- npm or yarn installed  

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/water-potability.git

# Navigate to project folder
cd water-potability

# Install dependencies
npm install

# Start the development server
npm run dev
